Math Coding Problems - Interview Quetsions

Some common math coding exercises used as interview programming questions

isPrime

An algorigthm to compute a Prime Number which is greater than 1 and only divisbile by one anditself. For example Prime numbers are 1,2,3,5,7,11,13,17..

Given a number n the function must determine if n is prime and return true/false.

boolean isPrime(int n) {
  for (int x=2; x*x <=n; x++) {
    if (n % x == 0) {
      return false;
    }
  }
  return true;
}

Analyzing this code fragment has a time complexity of o(n/2)
